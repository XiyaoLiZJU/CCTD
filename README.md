# CCTD
The first non-heading Chinese cabbage trichome dataset was constructed called CCTD, which contained 10,955 RGB images and 120,331 trichomes. Dataset images obtained from a binocular microscope. The image resolution was 3000Ã—4000 and saved in JPG format.
Examples of dataset images  were located in the folder 'Examples'.
The non-heading Chinese cabbage trichome dataset and manual annotation file (VOC format) are located in the link: https://pan.baidu.com/s/1ClAbmG7yElpsY1XddJWASA 
The extraction code is tdab.
